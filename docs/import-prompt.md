# Промпт для генерации импорта

Скопируй текст ниже целиком, вставь в любую нейронку (Claude, ChatGPT и т.п.), а следом — сырой текст источника (пост из Instagram, статья, переписка и т.д.). Нейронка соберёт markdown-файл, который можно сразу вставить на экран «Импорт» в приложении.

---

Ты помогаешь собрать markdown-файл для импорта идеи свидания в личное приложение. Ниже — сырой текст (например, пост из Instagram) с описанием места или мероприятия в Москве. Разбери его и верни **только markdown** в следующем формате, без пояснений до или после:

```markdown
# Название свиданки
Тип: DATE
Теги: тег1, тег2
Цена: ...
Описание для свайпа: короткий продающий текст для карточки

Место:
Адрес: ...
Метро: ...
Координаты: 55.75, 37.61
Ссылка: ...

Место:
Адрес: ...
Метро: ...
Координаты: 55.75, 37.61
Ссылка: ...

Свободное описание в несколько строк, если есть что добавить.
```

Правила:
- `# Название` — обязательно, одна строка, коротко и по сути.
- `Тип` — `DATE` для свиданок или `FOOD` для кафе, ресторанов, баров и других заведений. Если тип не указан, импорт считает запись свиданкой.
- `Теги`, `Цена`, `Описание для свайпа` — необязательны, включай только то, что реально есть в исходном тексте. Ничего не выдумывай.
- `Место:` — необязательный раздел для адреса. Если мест несколько, повторяй блок `Место:` для каждого адреса. Для одного места можно использовать старый короткий формат без строки `Место:`.
- `Адрес`, `Метро`, `Координаты`, `Ссылка` внутри блока `Место:` — необязательны. `Ссылка` — сайт, бронирование, Instagram, Яндекс/Google Карты или другой полезный URL.
- `Координаты` — только если в тексте явно даны координаты или очень конкретный адрес, из которого ты уверенно знаешь координаты; в формате `широта, долгота` (например `55.751244, 37.618369`). Если не уверен — не пиши поле вообще, лучше пусто, чем неверно.
- `Теги` — 2-5 слов через запятую, описывающих вайб/стиль (например: романтика, активити, искусство, еда, природа, ночная жизнь, бюджетно). Придумывай теги сам исходя из содержания поста.
- `Цена` — свободный текст как есть в источнике («бесплатно», «от 1500 ₽», «1000-3000 ₽ с человека»), если цена не указана — не пиши поле.
- `Описание для свайпа` — 1 короткое предложение без служебной инфы: суть идеи, ключевые плюсы и почему это хочется выбрать в свайпе.
- Описание (свободный текст после полей) — 1-3 предложения своими словами: что там происходит, почему может быть интересно на свидании. Не копируй хэштеги, эмодзи-простыни и рекламный мусор из исходного поста.
